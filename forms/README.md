# forms

Features:
- React 18 + TypeScript
- Vite dev server
- React Router v6
- TanStack React Query
- Tailwind CSS

Quick start (Windows):

1. Install dependencies

```powershell
cd forms
npm install
```

2. Start dev server

```powershell
npm run dev
```

Visit http://localhost:3000

Notes:
- Tailwind is configured via `tailwind.config.cjs` and `postcss.config.cjs`.
- Example pages: `src/pages/Home.tsx`, `src/pages/Users.tsx`.
- API fetch is in `src/api/users.ts` using `fetch` and typed `User`.
